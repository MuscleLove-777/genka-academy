# 原価計算アカデミー

見えないコストを見える化する、CRO原価計算の教科書。

## 機能

- **職種別単価ランキング**: CRO 8職種を単価・需要スコアでソート・フィルター・検索
- **CRO職種ガイド**: CRA/DM/統計解析/MW/PV/QA/PM/薬事の業務内容・スキル・1日のスケジュール
- **試験フェーズ別コスト**: Phase I〜PMSの費用構造・工数配分を可視化
- **原価計算シミュレーター**: 条件を入力してプロジェクトの原価・見積額をリアルタイム計算
- **原価計算クイズ**: 10問のクイズで原価計算マスター度を診断

## 技術構成

- HTML / CSS / JavaScript（フレームワーク不使用）
- レスポンシブ対応（スマホ・PC）
